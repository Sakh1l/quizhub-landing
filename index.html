// How it works

Up and running in four steps

From zero to live quiz in under two minutes.

01

Sign in as admin

Open the admin panel and enter the access code to unlock the full host dashboard.

02

Create your quiz

Add questions, answer choices, and configure session settings before going live.

03

Share & go live

Start the session and share the session code — participants join from any device instantly.

04

See live results

Scores update in real time. The leaderboard shows rankings after each round.
